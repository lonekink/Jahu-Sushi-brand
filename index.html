Brand guidelines.

We are very proud of our logo, and require that you follow these guidelines to ensure it always looks best. Our logo is the combination of a simple and modern woodmark with the typographic icon.

Download Logo & Icon

Using the logo

The Jahu Sushi salmon pink logo, pictured top left, is our primary logo colorway, and it should only be used with black, white, and non-duotoned photography.

Which color logo to use

The Jahu Sushi salmon pink logo should only be used on a light or white background, for any other background you should use a monochrome logo.

The black logo should be used on light colored backgrounds. The white logo should be used on dark colored backgrounds.

Exclusion zone

The logo and the icon’s exclusion zone is equal to half the height of the icon (marked as × in the diagram).

Minimum size

Establishing a minimum size ensures that the impact and legibility of the logo aren’t compromised.

Logotype should never be smaller than 70px in digital or 20mm in print.

Logo misuse

It’s important that the appearance of the logo remains consistent. The logo should not be misinterpreted, modified, or added to. Its orientation, color, and composition should remain as indicated in this document — there are no exceptions.

Using our colors

While embracing a much more colorful language in our brand communications, Jahu Sushi salmon pink is our resting color, used only in situations where the brand palette is not being used.

Jahu Sushi salmon pink should only ever sit on white, light grey, or a non-duotoned photograph. Don’t use Jahu Sushi salmon pink with a color from the brand palette or a duotoned image.

Fonts

What font to use?

We recommend you to use Roboto Mono for body text and Uniform Condensed for Headings, or at least default monospace font for the platform you are working on.

Using our typeface

If you want to use Jahu Sushi Uniform in your integration please seek approval via brand@jahu.ee.

You can find our web fonts here.

Using fallback fonts

In cases when you can’t use Uniform (or it doesn’t contain the characters needed), you have to chose one of our fallback fonts.

For example, you won’t be able to read “丸っこい” in Uniform, because it doesn’t contain Japanese Hiragana characters. So those displays using the default sans-serif font for whatever operating system you’re reading this on.

If you can’t use any custom fonts in your work, try commonly available defaults in this order:

  • Default sans-serif condensed for the platform
  • Roboto Condensed
  • Oswald
  • Fira Sans Compressed
